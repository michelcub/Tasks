## Getting Started
Install dependency
``` bash
yarn
```

First, run the development server:

```bash
yarn run dev

```

Init Prisma
```bash
yarn prisma init --datasource-provider sqlite
```
```bash
yarn prisma migrate dev --name init
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
